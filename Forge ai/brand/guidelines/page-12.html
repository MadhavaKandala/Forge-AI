Usage

  • Default typeface for all UI — labels, body, data, navigation
  • Tabular numerals for anything counted: reps, weight, days, time
  • Uppercase + wide tracking signals "system" — labels, statuses, tags
  • Sentence case for conversational copy (story, mission, alerts)

Don't

  • Don't mix in another sans/serif typeface anywhere
  • Don't use proportional numerals for streaks or stats
  • Don't set body copy in all caps — reserve caps for labels
  • Don't go below 11px — this is a workshop, not a spec sheet
